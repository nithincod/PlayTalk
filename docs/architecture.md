# Sports AI Commentary System (Badminton)

## High-Level Flow
Admin → Backend → Firebase → Flutter App

## Badminton Match Flow
MATCH_START → RALLY_START → SHOT_PLAYED → POINT_WON → SCORE_UPDATE → ...


## Description
This system generates live AI-based commentary for a badminton match.
Match events are sent to the backend, stored in Firebase, and streamed
in real time to the Flutter application.
